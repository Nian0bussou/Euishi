# Usage :

## Commands:

    move      sort the files

    scramble  scramble the files

    remove    remove `.tmp` files

    help      Print this message or the help of the given subcommand(s)


## Options of `move` & `scramble`:

    -p, --path <PATH>                provide the path

    -h, --help                       Print help

## Options of `remove`:

    -p, --path <PATH>  provide the path

    -v, --verbose

    -h, --help         Print help
